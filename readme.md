## üì¶ Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/bizzy-coding/React-Airbnb-Clone.git
   ```
2. Navigate to the project directory:
   ```sh
   cd React-Airbnb-Clone
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
